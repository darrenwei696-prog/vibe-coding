# 邮件助理 (comcom)

一个简洁高效的批量邮件处理工具，帮助您快速将通用邮件内容发送给不同的收件人。

## 功能特点

- 📧 **批量处理** - 一次性处理多封邮件，自动替换收件人信息
- 📊 **表格导入** - 支持上传Excel表格导入收件人列表
- 📄 **Word文档解析** - 上传Word文档作为邮件内容，自动识别黄色高亮内容为需要替换的部分
- 🎯 **主题提取** - 自动从Word文档中提取红色背景内容作为邮件主题
- 🔐 **SMTP配置** - 支持多种邮箱服务商的SMTP设置，确保邮件可靠发送
- 📋 **结果展示** - 详细的处理结果统计和状态展示

## 技术栈

- React 18+
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion (动画效果)
- Sonner (通知提示)

## 快速开始

### 前提条件

- Node.js 16+
- pnpm (推荐的包管理器)

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

应用将在 http://localhost:3000 启动

### 构建生产版本

```bash
pnpm build
```

构建后的文件将位于 `dist` 目录中

## 使用指南

1. **填写发件人邮箱** - 输入您的邮箱地址，系统会自动填充对应的SMTP设置
2. **配置SMTP设置** - 展开SMTP设置面板，确认服务器和端口设置，并输入邮箱授权码
3. **上传收件人表格** - 上传包含"名称"和"邮箱"两列的Excel表格
4. **上传邮件内容文档** - 上传Word文档，其中:
   - 黄色高亮的文本将被替换为收件人姓名
   - 红色背景的文本将被自动识别为邮件主题
5. **点击"开始处理"** - 系统将开始批量处理邮件
6. **查看处理结果** - 页面会显示每个收件人的处理状态和详细信息

## 注意事项

- 确保您的邮箱已开启SMTP服务并获取了授权码
- 不同邮箱提供商的SMTP设置和授权码获取方式可能不同，请参考内置的指引
- 当前应用支持的文件格式:
  - 收件人表格: .xls, .xlsx
  - 邮件内容: .doc, .docx
- 为避免邮件被标记为垃圾邮件，建议控制发送频率和数量
